## Contexto Visual da Marca

### Paleta de Cores

| Cor | HEX | Uso |
|-----|-----|-----|
| Roxo Primario | `#7C3AED` | Cor principal da marca |
| Roxo Escuro | `#5B21B6` | Sombras, overlays |
| Ciano Secundario | `#06B6D4` | Acentos, destaques |
| Amarelo Destaque | `#FACC15` | CTAs, elementos especiais |
| Background | `#020617` | Fundo escuro |

### Estilo Visual

- **Estetica**: Dark mode, neon, tech moderno
- **Tema**: Cultura geek, games, tecnologia, fantasia, sci-fi
- **Tom**: Vibrante mas sofisticado, nerd com classe
- **Abordagem**: Minimalista com elementos iconicos

### Regra Importante: Sem Texto nas Imagens

> **CRITICO**: Todas as imagens geradas devem ser **100% livres de texto**.
> Nao incluir: palavras, letras, numeros, logos, marcas, simbolos textuais, watermarks.
> O texto sera adicionado posteriormente via HTML/CSS no site.

## 1. Imagem do Header da Categoria (Banner)

### Especificacoes Tecnicas

| Propriedade | Valor |
|-------------|-------|
| Tamanho | 1920 x 400 px |
| Aspecto | 4.8:1 (wide banner) |
| Formato | JPG/WebP |
| Uso | Fundo do header da pagina de categoria |

### Template Base - Midjourney

```
Wide banner image for "[NOME_DA_CATEGORIA]" category page on a geek gift website.

Theme: [DESCRICAO_DA_CATEGORIA]

Visual style:
- Dark moody background with subtle purple (#7C3AED) and cyan (#06B6D4) lighting
- Cinematic composition, wide shot
- Thematic objects/elements related to [NOME_DA_CATEGORIA] scattered artistically
- Atmospheric fog or particle effects
- Professional product photography aesthetic
- Left side slightly darker for text overlay readability

Mood: Epic, immersive, premium geek aesthetic
Quality: Ultra detailed, 8K, photorealistic with stylized lighting

IMPORTANT: No text, no words, no letters, no numbers, no logos, no watermarks, no writing of any kind in the image.

--ar 48:10 --v 6 --style raw --q 2 --no text words letters numbers logos watermarks writing typography
```

### Template Base - DALL-E 3

```

Category description: [DESCRICAO_DA_CATEGORIA]

1. Dark, moody background with deep purples and subtle cyan accent lighting
2. Feature iconic items/elements that represent [NOME_DA_CATEGORIA]
3. Cinematic depth of field with atmospheric effects
4. Objects should be arranged artistically, not cluttered
5. Left portion should be slightly darker/simpler for text overlay

Style: Photorealistic with dramatic studio lighting, editorial product photography meets sci-fi movie poster

CRITICAL: Do not include any text, words, letters, numbers, logos, brand names, watermarks, or writing of any kind in the image. The image must be completely text-free.
```

## 3. Dicas para Melhores Resultados

### Parametros Recomendados por Ferramenta

#### Midjourney
- `--v 6` ou `--v 6.1`: Versao mais recente
- `--style raw`: Menos estilizacao artistica, mais fiel ao prompt
- `--ar 48:10`: Para banners (aproxima 1920x400)
- `--ar 1:1`: Para cards quadrados
- `--q 2`: Qualidade maxima
- `--no text, words, letters`: Evitar texto gerado

#### DALL-E 3
- Especificar "no text in the image"
- Descrever aspect ratio no prompt
- Usar termos como "photorealistic", "professional photography"
- Mencionar cores em HEX para precisao

#### Leonardo.ai
- Usar negative prompts ativamente
- Especificar "no text, words, letters" sempre
- Modelo recomendado: Leonardo Diffusion XL ou Leonardo Vision XL

#### Google Imagen (ImageFX)
- Usar linguagem natural e descritiva (sem parametros tecnicos)
- Descrever a cena como se estivesse narrando uma fotografia
- Incluir cores em HEX entre parenteses para precisao
- Especificar aspect ratio por descricao ("square", "ultra-wide panoramic")
- Enfatizar "no text" de forma clara e natural no final do prompt
- Acessar via: https://aitestkitchen.withgoogle.com/tools/image-fx
- Dica: ImageFX responde bem a descricoes de iluminacao e atmosfera

### Palavras-Chave Uteis

**Para atmosfera geek**:
- neon lighting, RGB glow, cyberpunk aesthetic
- gaming setup, collector's display
- sci-fi atmosphere, futuristic
- pixel art elements, retro gaming

**Para qualidade premium**:
- professional product photography
- studio lighting, dramatic shadows
- ultra detailed, 8K, high resolution
- editorial style, luxury aesthetic

**Para composicao**:
- cinematic composition, wide shot
- centered hero object, minimal background
- atmospheric fog, particle effects
- dark moody background, gradient lighting

## 4. Checklist de Validacao

Antes de usar a imagem gerada, verifique:

### Para Header (Banner)
- [ ] Aspecto correto (aproximadamente 1920x400 ou 4.8:1)
- [ ] Area escura a esquerda para texto
- [ ] Sem texto/letras na imagem
- [ ] Cores alinhadas com a marca (roxo/ciano)
- [ ] Tema reconhecivel da categoria
- [ ] Qualidade adequada (sem artefatos)

### Para Subcategoria (Card)
- [ ] Aspecto correto (quadrado 1:1)
- [ ] Objeto central claro e reconhecivel
- [ ] Area escura na parte inferior para label
- [ ] Sem texto/letras na imagem
- [ ] Cores alinhadas com a marca (roxo/ciano)
- [ ] Funciona bem em tamanho 400x400
