## Tipografia do Logo: Bungee

### Sobre a Fonte

| Característica | Descrição |
|----------------|-----------|
| **Nome** | Bungee |
| **Designer** | David Jonathan Ross (DJR) |
| **Estilo** | Display/urbana, signage, bold |
| **Inspiração** | Letreiros urbanos, neon, murais, marquises de cinema |
| **Licença** | SIL Open Font License (gratuita e open source) |
| **Fonte** | [Google Fonts](https://fonts.google.com/specimen/Bungee) |

### Características Técnicas

| Aspecto | Descrição |
|---------|-----------|
| **Forma** | Bold, geométrica, lados retos em curvas |
| **Legibilidade** | Alta - formas bem definidas funcionam em tamanhos pequenos |
| **Orientação** | Funciona horizontal E vertical (tipografia vertical nativa) |
| **Camadas** | Disponível em Regular, Inline, Outline e Shade |

### Família Bungee Disponível

| Variante | Uso Recomendado |
|----------|-----------------|
| **Bungee Regular** | Logo principal, texto sólido |
| **Bungee Inline** | Efeitos decorativos, destaques |
| **Bungee Outline** | Versões alternativas, contornos |
| **Bungee Shade** | Efeitos 3D, sombras |
| **Bungee Spice** | Versões com gradientes coloridos (COLRv1) |

### Por Que Escolhemos Bungee

1. **Identidade urbana/geek**: Estética que ressoa com cultura pop e gaming
2. **Alta versatilidade**: Funciona em horizontal e vertical nativamente
3. **Legibilidade superior**: Formas geométricas claras em qualquer tamanho
4. **Sistema de camadas**: Permite criar efeitos visuais (3D, outline, etc.)
5. **Open source**: Gratuita, sem custos de licenciamento
6. **Impacto visual**: Bold e memorável, ideal para branding

## Regras de Uso

### DO (Faça)
- Mantenha as proporções originais do logo
- Use fundo contrastante quando usar versão transparente
- Deixe espaço em branco ao redor do logo (mínimo 10% da altura)
- Use versões com fundo quando transparência não for suportada
- Mantenha o texto sempre em Bungee para consistência
- Utilize as variantes da família Bungee (Outline, Shade) para efeitos especiais aprovados

### DON'T (Não Faça)
- Não distorça ou estique o logo
- Não altere as cores do logo sem aprovação
- Não substitua a fonte Bungee por outra tipografia
- Não adicione efeitos não aprovados (sombras, brilhos aleatórios)
- Não coloque o logo sobre fundos muito coloridos ou com padrões
- Não use o logo em tamanhos menores que 32px de altura
- Não separe o mascote do texto de forma que percam a associação

## Implementação da Fonte Bungee

### CSS para Web

```css
/* Importação via Google Fonts */
@import url('https://fonts.googleapis.com/css2?family=Bungee&display=swap');

/* Uso no CSS */
.logo-text {
  font-family: 'Bungee', cursive;
  font-weight: 400;
  font-style: normal;
  color: #F5B81C;
}
```

### HTML

```html
<!-- No <head> -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Bungee&display=swap" rel="stylesheet">
```

### Variantes Adicionais (quando necessário)

```css
/* Todas as variantes da família Bungee */
@import url('https://fonts.googleapis.com/css2?family=Bungee&family=Bungee+Inline&family=Bungee+Outline&family=Bungee+Shade&family=Bungee+Spice&display=swap');
```
