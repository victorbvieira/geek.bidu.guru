## Estrutura Sugerida

Pillar page (guia completo) + 5–10 cluster pages por tópico.

Clusters principais:
- Por Ocasião: Natal, Aniversário, Amigo Secreto, Namorados
- Por Perfil: Gamers, Devs, Otakus, Boardgamers
- Por Preço: até R$ 50, R$ 50–100, R$ 100–200, +R$ 200
- Por Categoria: Decoração, Tecnologia, Utilidades, Colecionáveis

---

## Regras

- Cada cluster page linka para a pillar e para 2–3 pages irmãs.  
- A pillar lista e contextualiza todas as cluster pages (ItemList + links).  
- Breadcrumbs consistentes (Home > Cluster > Post).  
- Atualização trimestral das pillars; semestral dos clusters.

---

## Mapeamento de URLs (exemplo)

`/hubs/presentes-geek/` (pillar geral)
- `/hubs/presentes-geek/ocasioes/` → lista de ocasiões  
- `/hubs/presentes-geek/perfis/` → lista de perfis  
- `/hubs/presentes-geek/preco/` → faixas de preço  

Cada item aponta para listicles/guides aprofundados.
