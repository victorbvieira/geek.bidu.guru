Como usar

No cliente (agente paperclip, n8n, curl), envie: 

Authorization: Bearer pcat_…

O servidor aceita tanto JWT (login normal) quanto API tokens (prefixo pcat_) no mesmo header. Para agentes de longa duracao, prefira API token.
