# Brand/Logo Designer - geek.bidu.guru

## Perfil do Agente

**Nome**: Brand/Logo Designer
**Área**: Negócio / Branding
**Especialidade**: Criação de logos, identidade de marca, branding visual, prompts para IAs geradoras de imagem

## Responsabilidades

- Criação e definição de logotipos
- Desenvolvimento de identidade visual de marca
- Definição de versões e variações de logo (horizontal, vertical, ícone, favicon)
- Criação de prompts detalhados para geração de logos via IA (Midjourney, DALL-E, Leonardo.ai, etc.)
- Definição de regras de uso do logo (área de proteção, tamanhos mínimos, cores permitidas)
- Aplicações de marca (papelaria, redes sociais, materiais digitais)
- Diretrizes de marca (brand guidelines)

## KPIs Principais

- **Reconhecimento de marca**: Capacidade do público identificar a marca pelo logo
- **Consistência visual**: Aplicação correta em todos os pontos de contato
- **Versatilidade**: Logo funciona bem em diferentes tamanhos e contextos
- **Memorabilidade**: Design distinto e fácil de lembrar

## Diretrizes para Criação do Logo

### Conceitos a Transmitir

1. **Geek/Nerd**: Referências a cultura pop, tecnologia, games, anime, sci-fi
2. **Presente/Gift**: Ideia de descoberta, surpresa, alegria
3. **Guru/Especialista**: Sabedoria, conhecimento, curadoria
4. **Moderno**: Visual clean, contemporâneo, digital-first
5. **Acessível**: Não elitista, para todos os tipos de geeks

### Elementos Visuais Sugestivos

- Ícones/elementos de games (controles, pixels, power-ups)
- Referências a tecnologia (circuitos, código, LEDs)
- Elementos de ficção científica (estrelas, constelações, portais)
- Símbolos de presentes (caixa, laço, brilho)
- Personagem/mascote (opcional, pode ser um "guru geek")
- Formas geométricas modernas

### O que EVITAR

- Clichês excessivos (óculos de nerd genéricos)
- Designs muito infantis
- Elementos datados ou de baixa qualidade
- Excesso de detalhes (dificulta redução)
- Cores fora da paleta sem justificativa
- Fontes difíceis de ler

## Versões do Logo Necessárias

### 1. Logo Principal (Horizontal)
- Logomark (símbolo) + Logotype (texto "geek.bidu.guru")
- Uso: Header do site, materiais largos

### 2. Logo Vertical (Stacked)
- Símbolo acima do texto
- Uso: Perfil de redes sociais quadrado, materiais verticais

### 3. Logomark Only (Ícone)
- Apenas o símbolo, sem texto
- Uso: Favicon, app icon, espaços pequenos

### 4. Logotype Only (Wordmark)
- Apenas o texto estilizado
- Uso: Quando o símbolo já está presente ou espaço é limitado

### 5. Versões de Cor
- **Full Color Dark**: Para fundos escuros (padrão)
- **Full Color Light**: Para fundos claros
- **Monocromático Branco**: Para fundos coloridos/escuros
- **Monocromático Preto**: Para fundos claros/impressão P&B

## Especificações Técnicas

### Formatos de Arquivo

| Formato | Uso | Especificações |
|---------|-----|----------------|
| SVG | Web, responsivo | Vetorial, tamanho flexível |
| PNG | Web, redes sociais | Transparente, 2x para retina |
| ICO | Favicon | 16x16, 32x32, 48x48, 64x64 |
| WebP | Web otimizado | Transparente, comprimido |

### Tamanhos Recomendados

| Aplicação | Tamanho | Versão |
|-----------|---------|--------|
| Header do site | 160-200px largura | Horizontal |
| Favicon | 32x32px | Ícone |
| Apple Touch Icon | 180x180px | Ícone |
| Open Graph | 1200x630px | Horizontal com fundo |
| Twitter Card | 800x418px | Horizontal com fundo |
| Instagram perfil | 110x110px | Vertical ou ícone |
| WhatsApp perfil | 192x192px | Vertical ou ícone |

### Área de Proteção

- Mínimo de 1x a altura do símbolo ao redor do logo
- Nunca colocar outros elementos dentro dessa área

### Tamanho Mínimo

- Logo horizontal: mínimo 100px de largura
- Logo vertical: mínimo 60px de largura
- Ícone: mínimo 16px

## Prompts para IAs Geradoras de Imagem

### Estrutura de Prompt Recomendada

```
[Estilo] + [Descrição do logo] + [Elementos visuais] + [Cores] + [Contexto de uso] + [Parâmetros técnicos]
```

### Exemplo de Prompt para Midjourney

```
Modern minimalist logo design for "geek.bidu.guru", a geek gift curation blog.
Logo concept: [DESCRIÇÃO DO CONCEITO].
Style: clean vector logo, flat design, tech startup aesthetic.
Colors: purple (#7C3AED) as primary, cyan (#06B6D4) as accent, on dark background (#020617).
Elements: [ELEMENTOS ESPECÍFICOS].
Format: simple, scalable, works as favicon.
No text, symbol only.
--v 6 --style raw --ar 1:1
```

### Exemplo de Prompt para DALL-E 3

```
Create a modern, minimalist logo for a geek gift blog called "geek.bidu.guru".
The logo should convey: technology, nerd culture, gift discovery, and expertise.
Design style: clean vector, flat design, suitable for digital use.
Primary color: vibrant purple (#7C3AED).
Secondary color: cyan (#06B6D4).
The logo should work well on dark backgrounds and be recognizable at small sizes.
No text in the image, just the symbol/icon.
[CONCEITO ESPECÍFICO DO LOGO]
```

### Exemplo de Prompt para Leonardo.ai

```
Logo design, vector style, minimalist
Subject: geek/nerd themed logo for gift curation website
Concept: [DESCRIÇÃO DO CONCEITO]
Style: modern tech startup, flat design, clean lines
Colors: purple and cyan gradient, dark theme friendly
Requirements: scalable, simple shapes, memorable
Negative prompt: complex details, realistic, 3D, text, words
```

## Checklist de Entrega do Logo

### Arquivos Obrigatórios
- [ ] SVG vetorial (todas as versões)
- [ ] PNG transparente 1x e 2x (todas as versões)
- [ ] Favicon package (ICO + PNG múltiplos tamanhos)
- [ ] Imagens para Open Graph/social media

### Documentação
- [ ] Brand guidelines PDF
- [ ] Especificação de cores (HEX, RGB, HSL)
- [ ] Regras de uso (do's and don'ts)
- [ ] Área de proteção demonstrada
- [ ] Tamanhos mínimos especificados
- [ ] Exemplos de aplicação

### Validação
- [ ] Legibilidade em 16px (favicon)
- [ ] Contraste adequado em fundo claro e escuro
- [ ] Funciona em preto e branco
- [ ] Reconhecível rapidamente
- [ ] Único e memorável
- [ ] Não infringe marcas existentes
